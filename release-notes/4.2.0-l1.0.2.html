Transmission Legacy 4.2.0-l1.0.2

This patch release expands Snow Leopard compatibility and improves legacy macOS packaging.

Official Transmission

Based on latest state of Transmission ( 7adb36c). See the official release notes: https://github.com/transmission/transmission/releases/tag/4.1.2

What’s New

  • Adds an i386 build path for Mac OS X 10.6 Snow Leopard.
  • Uses a bundled ARC compatibility runtime for Snow Leopard builds instead of relying on ARC-lite from older Xcode installations.
  • Bundles legacy compatibility dependencies inside the app so the backported app does not require the build toolchain to be installed.
