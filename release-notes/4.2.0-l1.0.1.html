Transmission Legacy 4.2.0-l1.0.1

This patch release includes compatibility and maintenance improvements for legacy macOS releases.

Official Transmission

Based on latest state of Transmission ( 8c9bc1b7). See the official release notes: https://github.com/transmission/transmission/releases/tag/4.1.2

What’s New

  • Fixes a crash on Mac OS X 10.6 when opening or editing group auto-assign rules from the Preferences window.
  • Fixes a Snow Leopard crash that could occur when adding a new group after earlier add-window or torrent activity.
  • Fixes a crash on Mac OS X 10.6 and OS X 10.7 when creating a torrent from the Create Torrent window.

Legacy macOS

  • Improves Snow Leopard compatibility for the group rules sheet wiring.
  • Brings the Snow Leopard and Lion Create Torrent window XIB back in line with the modern creator UI, including Source and piece-size controls.
